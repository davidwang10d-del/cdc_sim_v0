# 🌍 CDC Sim v0 - 军工级地理空间兵棋引擎

![License](https://img.shields.io/badge/License-PolyForm%20Noncommercial-blue.svg)
![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![Status](https://img.shields.io/badge/Status-Alpha%20Release-success)

**CDC Sim v0** 是一个开源的、基于 Web 的军工级战术大战略兵棋推演引擎。它采用 Python (Flask后端) 与纯原生 JS (Leaflet前端) 构建，将真实的地理信息系统 (GIS) 与动态战斗物理法则相结合。

本项目采用完全解耦的数据驱动架构，支持热插拔剧本模组 (DLC)，并内置了功能强大的 GUI 剧本编辑器，旨在为军迷、推演爱好者和研究人员提供一个高度自由的沙盘模拟平台。

---

## ✨ 核心特性 (Core Features)

* **🗺️ 真实的地理信息系统 (GIS)**
  * **90m DEM 高程解析**：基于 `rasterio` 实时读取全球高程数据，实现真实的高地火力压制与移动速度惩罚。
  * **3D 视距计算 (LOS)**：基于 Bresenham 射线追踪算法，结合地球曲率计算真实的雷达探测盲区。
* **⚔️ 动态战斗与物理引擎**
  * **半正矢后勤与机动**：基于 Haversine 公式计算球面距离，结合兵种属性（步兵、装甲、空降）演算真实行军时间。
  * **士气与溃散机制**：模拟部队交火时的组织度损失、防线崩溃与自动撤退。
* **🛠️ 终极模组化 (Mod-Friendly)**
  * **全数据驱动**：引擎本身不包含任何特定战役数据。所有地图边界、势力、建制、武器参数全部通过 JSON 动态挂载。
  * **GUI 剧本架构师**：内置可视化数据库编辑器，无需写代码即可创造新战役。
* **⚡ 实时干预与管理**
  * **上帝模式 (Zeus Console)**：允许裁判/指挥官实时修改部队状态、强行介入外交关系。
  * **态势存档系统 (Save/Load)**：支持多槽位实时态势记忆与一键时光倒流。

## 🚀 快速上手 (Quick Start)

### 1. 环境准备 (Prerequisites)
请确保你的电脑已安装 **Python 3.10** 或更高版本。

克隆本仓库到本地：
```bash
git clone [https://github.com/davidwang10d-del/cdc_sim_v0.git](https://github.com/davidwang10d-del/cdc_sim_v0.git)
cd cdc_sim_v0
安装所需依赖：

Bash
pip install -r requirements.txt
(主要依赖包括：Flask, rasterio, numpy, Pillow)

2. 准备地形数据 (可选但推荐)
为了激活 3D 高程视距功能，你需要在对应的 DLC 文件夹（如 dlc/afghan/maps/dem_tiles/）中放入 SRTM 90m 分辨率的 .tif 高程贴图。
(如果未放置高程图，引擎将自动降级为 2D 平面推演模式)

3. 启动引擎
在终端中运行以下命令启动中央服务器：

Bash
python server.py
终端显示 Running on http://127.0.0.1:5000 后，打开浏览器访问 http://localhost:5000 即可进入指挥官终端。

📖 完整使用教学 (User Manual)
模块一：启动与全局控制
中英双语切换：进入系统后，点击右上角的 中文 或 English 即可全局切换界面语言。

挂载战区：在黑色的启动界面（Bootloader）中，系统会自动扫描 dlc/ 目录下的可用战役。点击对应战役即可将其挂载至物理引擎并进入雷达界面。

时间流逝：顶部控制台包含时间控制按钮。点击 ▶ 启动推演时间，点击 ⏸ 暂停。你可以通过设定 1X, 5X 或自定义倍率来加速推演。

模块二：战术指挥与部队调度
查看情报：在地图上点击任何带有图标的部队，右下角的「高精度遥测雷达」将显示该部队的建制构成、士气、武器射程及当前状态。

下达移动/攻击指令：

选中己方部队。

在地图上的任意目标点双击。

屏幕中央将弹出「战术授权确认」面板，确认坐标无误后点击“✅ 授权执行”。部队将立即规划路线并开拔。

空投部署 (Airdrop)：

点击顶部控制台的 🚁 空投 按钮。

在左侧弹出的面板中输入新部队的番号、阵营、图标（支持 Emoji）以及建制兵力。

点击“📍 在地图上选择空投坐标”，随后在地图上点击目标位置，增援部队将瞬间抵达战场。

模块三：态势存档与上帝模式
档案管理 (Save/Load)：

点击顶部的 💾 档案 按钮打开管理面板。

写入：在发起重大军事行动前，选择一个槽位（如 Slot 1）点击“写入”，系统将保存当前全部态势。

读取/重置：战局不利时，选择对应槽位点击“载入”即可回档。选择 Initial State (Hard Reset) 可将战役完全重置为初始状态。

上帝干预 (Zeus)：

点击顶部的 ⚡ 上帝 按钮。

选中地图上的任意部队，你可以在此面板中瞬间抽干其兵力、修改其战斗效率（BUFF/DEBUFF），或者强行修改两个阵营之间的外交状态（和平/宣战）。

🔧 剧本架构师 (GUI 模组编辑器)
CDC Sim v0 内置了一个强大的可视化编辑器，让你轻松创建属于自己的战争史诗。

在游戏主界面点击顶部控制台的 🔧 编辑器，或直接访问 http://localhost:5000/editor。

在左侧边栏导航中，你可以直观地修改：

战役全局设置：地图边界经纬度、战役背景描述。

战斗序列 (OOB)：在地图上预置初始部队，设置其自定义雷达图标（如 🚁、💀）。

物理与规则：自定义不同兵种（装甲、步兵）的杀伤力、速度，以及各种武器的有效射程。

编辑完成后，点击右上角的 “💾 导出 DLC 全量数据包”。系统将自动生成一个 .zip 文件，解压并将其放入 dlc/ 目录下，即可作为新战区运行！
